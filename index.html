Introduction

This example demonstrates the basic structure of an HTML5 page that is accessible and SEO-friendly without using CSS or JavaScript.

Key Features

Semantic Elements

Using tags like <header>, <nav>, <main>, and <footer> provides meaning to the page structure.

Accessibility

ARIA attributes such as aria-labelledby and aria-label enhance screen reader navigation.

SEO Basics

Proper heading hierarchy and metadata help search engines understand and rank content effectively.

Conclusion

This simple HTML5 page adheres to modern web standards by being semantically structured, accessible, and SEO-aware—without any extra styling or scripting.
